## Dataset Source and Credits

This project makes partial use of the dataset published in:

Al-Emadi, S. A., Al-Ali, A. K., Al-Ali, A., & Mohamed, A. (2019).  
**Audio Based Drone Detection and Identification using Deep Learning**.  
IWCMC 2019 Vehicular Symposium (IWCMC-VehicularCom 2019), Tangier, Morocco.

The original dataset was used strictly as a complementary resource.  
The dataset included in this repository also contains **original audio recordings collected and processed by the author**.

All rights and credits for the original dataset remain with the original authors.

**Instituto Politécnico Nacional (IPN)**

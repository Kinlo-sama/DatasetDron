## Dataset Source and Credits

This project makes partial use of datasets and methodologies reported in the following works:

Al-Emadi, S. A., Al-Ali, A. K., Al-Ali, A., & Mohamed, A. (2019).  
**Audio Based Drone Detection and Identification using Deep Learning**.  
IWCMC 2019 Vehicular Symposium (IWCMC-VehicularCom 2019), Tangier, Morocco.

Akbal, E., Akbal, A., Dogan, S., & Tuncer, T. (2023).  
**An automated accurate sound-based amateur drone detection method based on skinny pattern**.  
Digital Signal Processing, 136, 104012. https://doi.org/10.1016/j.dsp.2023.104012

The dataset published by Al-Emadi et al. (2019) was used strictly as a complementary resource.  
The dataset described by Akbal et al. (2023) is referenced as a methodological and comparative source; no publicly available dataset citation or direct dataset access is provided by the authors.

The dataset included in this repository also contains **original audio recordings collected and processed by the author**.

All rights and credits for third-party data and methodologies remain with the original authors.
